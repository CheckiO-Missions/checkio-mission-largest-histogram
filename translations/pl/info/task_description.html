Mają Państwo histogram. Proszę spróbować znaleźć rozmiar największego prostokąta, jaki można zbudować ze słupków histogramu.

Dane wejściowe: {% if interpreter.slug == "js-node" %}Tablica{% else %}Lista{% endif %} wysokości wszystkich prostokątów na histogramie.

Dane wyjściowe: Pole największego prostokąta.

Przykład:

{% if interpreter.slug == "js-node" %} 
largestHistogram([5]) == 5
largestHistogram([5, 3]) == 6
largestHistogram([1, 1, 4, 1]) == 4
largestHistogram([1, 1, 3, 1]) == 4
largestHistogram([2, 1, 4, 5, 1, 3, 3]) == 8
{% else %} 
largest_histogram([5]) == 5
largest_histogram([5, 3]) == 6
largest_histogram([1, 1, 4, 1]) == 4
largest_histogram([1, 1, 3, 1]) == 4
largest_histogram([2, 1, 4, 5, 1, 3, 3]) == 8
{% endif %}

Sposób użycia: Nie ma możliwości, aby rozwiązanie, które Państwo wymyślili, było przydatne w prawdziwym życiu. Proszę się dobrze bawić!

Warunek wstępny:
0 < len(data) < 1000
