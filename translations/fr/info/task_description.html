Vous avez un histogramme. Essayez de trouver la taille du plus grand rectangle que vous pouvez construire à partir des barres de l'histogramme.

Entrée: {% if interpreter.slug == "js-node" %}Un tableau{% else %}Une liste{% endif %} des hauteurs de tous les rectangles de l'histogramme.

Résultat: Surface du plus grand rectangle.

Exemple:

{% if interpreter.slug == "js-node" %} 
largestHistogram([5]) == 5
largestHistogram([5, 3]) == 6
largestHistogram([1, 1, 4, 1]) == 4
largestHistogram([1, 1, 3, 1]) == 4
largestHistogram([2, 1, 4, 5, 1, 3, 3]) == 8
{% else %} 
largest_histogram([5]) == 5
